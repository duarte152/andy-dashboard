# Fulbright Brasil — Pesquisa de Status 2026

*Pesquisado em: 2026-04-17*

## Status Atual (abril/2026)

O Programa Fulbright no Brasil **não tem uma janela geral de aplicação** para mestrado. A Comissão Fulbright Brasil administra programas temáticos específicos.

| Programa | Status | Próxima abertura |
|---|---|---|
| Doutorado Sanduíche nos EUA (até 50 bolsas, via CAPES) | **Fechado** — encerrou set/2025 | **Previsto: junho 2026** |
| Mestrado em Roteiro de Cinema (MFA) | **Fechado** — encerrou abr/2025 | Previsto: abr/2027 |
| Humphrey Fellowship (profissionais mid-career, setor público/ONGs) | **Fechado** — encerrou jul/2025 | **Previsto: ~abr/2026** |
| PVCD 2026 – Fulbright Specialist | **ABERTO** | Para universidades, não indivíduos. Prazo: 29/mai/2026 |

## Conclusão para o Thiago

**Não há programa Fulbright de mestrado geral aberto para brasileiros em tecnologia/gestão/saúde agora.**

O programa mais próximo do perfil de Thiago seria o **Humphrey Fellowship** (para profissionais mid-career), mas ele é voltado para setor público/ONGs e exige 5 anos de experiência. O **Doutorado Sanduíche** é para quem já está num doutorado brasileiro.

**Ação recomendada:** monitorar fulbright.org.br/bolsas-para-brasileiros/ a partir de junho/2026 para novas aberturas.

## Onde monitorar

- https://fulbright.org.br/bolsas-para-brasileiros/ (filtrar "Abertos" + "Estudantes")
- https://foreign.fulbrightonline.org/apply (selecionar Brasil)
- Contato: +55 61 3248-8600 | Brasília

## Documentos Necessários (baseado em ciclos anteriores)

- Cidadania brasileira (sem cidadania americana)
- Diploma de graduação (mínimo 4 anos)
- TOEFL iBT 102+ ou IELTS 7.0+ (após junho/2024)
- Sem mestrado prévio (para bolsas de estudante)
- Residência no Brasil durante a aplicação
- Statement of purpose / proposta de pesquisa
- Cartas de recomendação
- Histórico acadêmico
- CV
